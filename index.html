Next, we multiply the first two rows by some values to make the first element in each row equal. This is done by taking the least common multiple of said elements and then multiplying each element by the least common multiple divided by itself.





We can now subtract the first row from the second to eliminate x in the second row.





We go through the same process as the first two rows to set the first element in rows 1 and 3 equal to each other.





We can now subtract the first row from the third to eliminate x in the third row.





We go through the same process as before for the first two rows again, this time setting the second number (y coefficient) of the second and third rows equal to each other, preparing another elimination.





We can now subtract the second row from the third to eliminate y in the third row.





We go through the same process as before for the first two rows again. This time, we want to eliminate z from the second row so we set the z coefficients in the last two rows equal to each other.





We can now subtract the third row from the second to eliminate z in the second row. Note that this does not affect x and y since those coefficients are zero in the third row.





We go through the same process as before for the first two rows for the final time, setting the y coefficients in the first and second row equal to each other.





We can now subtract the second row from the first to eliminate y in the first row.





We go through the same process as before for the first two rows again. This time, we want to eliminate z from the first row so we set the third element of the first and third rows equal to each other.





We can now subtract the third row from the first to eliminate z in the first row.
